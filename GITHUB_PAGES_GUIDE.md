# 茶马古道·春茶之旅 2026 - GitHub Pages 部署指南

## 🚀 一键部署步骤

### 步骤1：访问GitHub仓库设置
1. 打开浏览器，访问：https://github.com/sbfeeg-spec/tea-trip-2026
2. 点击右上角 "Settings"（设置）

### 步骤2：配置GitHub Pages
1. 在左侧菜单中找到 "Pages"（页面）
2. 在 "Build and deployment"（构建和部署）部分：
   - **Source**（源）：选择 "Deploy from a branch"（从分支部署）
   - **Branch**（分支）：选择 `main`，文件夹选择 `/(root)`
3. 点击 "Save"（保存）

### 步骤3：等待部署完成
1. GitHub会自动开始部署（约1-2分钟）
2. 刷新页面，你会看到绿色的部署成功消息
3. 你的网站地址：**https://sbfeeg-spec.github.io/tea-trip-2026/**

## 🔧 自定义设置（可选）

### 自定义域名
如果你想使用自己的域名（如 tea-trip-2026.com）：
1. 在Pages设置中找到 "Custom domain"（自定义域名）
2. 输入你的域名
3. 按照提示配置DNS

### 启用HTTPS
GitHub Pages自动提供免费的SSL证书，HTTPS默认启用。

### Q: 页面显示404错误？
A: 等待1-2分钟让部署完成，然后强制刷新浏览器（Ctrl+F5）。

### Q: 如何更新内容？

### Q: 可以添加更多页面吗？
